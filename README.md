# 鸿蒙 ArkTS 仿 YouTube Music app

## 介绍

基于 ArkTS 编写的适用于鸿蒙 HarmonyOS 的 YouTube Music app clone, 用于学习练手。

## 后端 API

后端 API 使用 Express.js 搭建，[GitHub 地址](https://github.com/Charles-Stark/yt-music-backend)。
